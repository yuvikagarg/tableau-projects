# Netflix Content Analysis Dashboard

## Overview

This Tableau dashboard provides an interactive analysis of Netflix's content library, helping users explore trends in movies and TV shows, content ratings, genres, release years, and country-wise distribution.

## Objectives

- Analyze Netflix content distribution.
- Identify popular genres and ratings.
- Understand content growth over time.
- Explore geographic distribution of content.

## Dashboard Features

- Movies vs TV Shows Distribution
- Content by Rating
- Top 10 Genres
- Content by Country
- Release Year Analysis
- Duration Analysis
- Date Added Trends

## Key Insights

- Comparison of Movies and TV Shows available on Netflix.
- Most common content ratings.
- Top-performing genres on the platform.
- Countries contributing the highest volume of content.
- Trends in content releases over time.

## Tools Used

- Tableau
- Excel/CSV Dataset
